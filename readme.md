## Getting Started

Clone the repo:
```sh
```

Install nodemon:
```js
npm install -g nodemon
```

Set environment (vars):
```sh
cp .env.example .env
```

Start server:
```sh
# Start server
npm run dev:server
```
Start client:
```sh
# Start server
npm run dev:client
```
```sh
# Build for Deployment
npm run dev:client
```

Tests:
```sh
# Run tests written in ES6 
npm test
```
